Avijat Shop BD - Mobile Setup

1. Download this ZIP.
2. Extract it using ZArchiver or Files app.
3. Open github.com and create a repository named avijatshopbd.
4. Upload all extracted files.
5. Go to Settings > Pages.
6. Source: Deploy from a branch.
7. Branch: main, Folder: /(root), Save.
8. Wait 2-5 minutes and open:
   https://YOUR-USERNAME.github.io/avijatshopbd
